Au cours de cette formation nous allons voir 2 matières :

L'intègration

Le HTML :

Le HTML sert à crée le contenu du site : elle va contenir uniquement la partie textuelle du site.

Le CSS :

Le CSS va nous permettre d'ajouter la couche graphique à notre site : les couleurs, les formes, le fonds, etc.

Le developpement

Le JavaScript :

JavaScript est un langage de programmation de script employe pour rendre une page web interactive, le comportement de la page et son apparance pourra varier en fonction de certain parametres.

Le PHP :

le PHP est un langage de programmation libre, utilisé pour genere des page web dynamiques, qui sont donc modifies en fonction de certaines informations (heure, nom de l'utilisateur, formulaire remplie par l'utilisateur etc.)

MySQL :

MySQL est un systeme de gestion de base de donnee, il permet d'enregistrer les informations rempli par l'utilisateur (exemple : formulaire de crèation de compte ) pour ensuite etre re-utilise au moment approprié (ex : connexion a un site internet avec identifiant et mot de passe)
